搭建博客的一些事儿

发表于 2014-05-02 分类于 Interest

五一放假，外面据说人山人海，也懒得出去。在家继续练习烹饪，编写代码，另外把想了些日子的独立博客搭了起来。

之前建过一个博客，买了个主机空间，部署了Wordpress，开始还有兴致，后来就慢慢懒得写了，没能坚持下去。写东西比想东西要难，脑袋里千回百转，可要用语言描述出来，却很耗时间和考验功力。

最近越来越希望跟人交流想法，分享自己的知识和经历，于是又生起了写博客的心思。不想再用之前的Wordpress，慢且繁冗，刚好发现了Farbox+Dropbox方案，不需要折腾空间，采用Markdown，简单纯粹，正符合现在的喜好。

搭建过程还是很简便的，网上有不少文章讲解，官网也有文档。

我在Godaddy上有两个域名，都是指向之前的那个博客，建了这个后，那个也就废弃了。大概是在大陆的原因，登Godaddy非常慢，管理起来很费劲。于是在Godaddy偶尔不抽风的时候，赶紧把域名的Nameserver都指向了国内的Dnspod，这样之后就可以在Dnspod上修改域名了。

Dnspod上要做的，就是建立CName，指向这个博客的Farbox地址。

域名修改后，生效需要一些时间。Dnspod服务器实时更新，但由于全球各地DNS服务器有缓存，短则几分钟，长则数小时才可用。要查看域名指向是否正确，可以用nslookup命令，Windows和Linux都支持。

上面的是查询的DNS服务器，我没有使用ISP默认的，而是使用了中华电信的168.95.192.1和168.95.192.2；下面是查询的结果，可以看到，域名qilu.me已经被正确解析成了metazion.farbox.com。

Linux下还可以用dig命令，有更详细的解析过程。

从上到下依次是查询根域名服务器，查询.me顶级域名服务器，查询Godaddy指向的Dnspod的Nameserver，查到qilu.me的CName记录metazion.farbox.com。

最后还有一件要做的事，就是在Farbox上绑定多域名，如本例可绑定qilu.me、www.qilu.me等，具体操作可以参考官方文档。
